Heroku เป็นบริการคลาวด์แบบ Platform as a Service (Paas) ที่ให้บริการสำหรับนักพัฒนาซอฟต์แวร์ สามารถรองรับภาษาต่างๆ เช่น Java , Python , PHP ,Ruby, Go และ Node.js เป็นต้น โดยนักพัฒนาซอฟต์แวร์ใช้ Heroku ในการ deploy และจัดการแอปพลิชัน เพราะ Heroku มี Add-ons สำหรับเพิ่มเติมบริการอื่นๆ เช่น PostgreSQL, MongoDB, Redis เป็นต้น มีให้เลือกใช้งานฟรี และเสียเงิน
